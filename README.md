# Snake Game

A classic snake game built with vanilla HTML, CSS, and JavaScript on an HTML5 canvas.

## How to Play

1. Open `snake/index.html` in your browser
2. Press any arrow key to start
3. Eat the red food to grow and earn points
4. Every 5 points the speed increases
5. Press **R** to restart after game over

## Features

- Retro green-on-black terminal aesthetic
- Increasing difficulty as your score grows
- High score saved in local storage
